## Getting Set Up: The Steps

Follow these steps to get your RSS feed up and running:

### Tools Required:

- [Microsoft Edge browser](https://www.microsoft.com/en-us/edge/download?form=MA13FJ)
- [FeedBro: RSS feed reader extension for Microsoft Edge](https://chrome.google.com/webstore/detail/feedbro/mefgmmbdailogpfhfblcnnjfmnpnmdfa?hl=en)
- [Upwork account](https://www.upwork.com)

### Step 1: Install Microsoft Edge

- Download [Microsoft Edge from the official website](https://www.microsoft.com/en-us/edge/download?form=MA13FJ).
- Follow the installation instructions to install the browser on your computer.

![Step 1](/static/blog/94/001.png)

### Step 2: Create a New User Profile on Edge

- Open Microsoft Edge and navigate to the upper-right corner of the browser window.
- Select the profile icon (it may look like a blank silhouette, a letter, or a custom image).
- Select "Manage profile settings" and then click "Add profile."
- Choose "Add" and decide whether to sync your data or create a new profile without data. If you choose to sync, sign in with your Microsoft account and confirm the settings.

![Step 2](/static/blog/94/002.png)

### Step 3: Install Grammarly Extension

- Open Microsoft Edge and navigate to the [Grammarly Chrome extension page](https://chrome.google.com/webstore/detail/grammarly-for-chrome/kbfnbcaeplbcioakkpcpgfkobkghlhen) on the Chrome Web Store.
- Click "Add to Chrome" to install the extension.
- A prompt will appear asking if you want to add the extension. Click "Add extension" to confirm.
- Once installed, you can access Grammarly by clicking its icon in the Edge toolbar and signing in or creating a new account.

![Step 3](/static/blog/94/003.png)

### Step 4: Install RSS Feed Reader Extension

- Open Microsoft Edge and navigate to the Edge Add-ons Store.
- Search for an RSS feed reader extension [RSS Feed Bro by FeedBro](https://chrome.google.com/webstore/detail/feedbro/mefgmmbdailogpfhfblcnnjfmnpnmdfa?hl=en).
- Click "Get" to install the extension.

![Step 4](/static/blog/94/004.png)

### Step 5: Configure RSS Feed Reader Settings

- Click on the RSS feed reader extension icon in the Edge toolbar.
- Go to "Settings" and make the following adjustments:
  - Set the Scan interval (default) to 5 minutes.
  - Enable the option to open the "All Items" page by default.
  - Choose your preferred theme (dark).

![Step 5](/static/blog/94/005.png)

### Step 6: Obtain Upwork RSS Feed URL

- Log in to your Upwork account and perform a job search using your desired keyword (check the Job Criteria and Keyword Section).
- Click on the RSS feed icon to generate the RSS feed URL.

![Step 6](/static/blog/94/006.png)

### Step 7: Copy the RSS Feed URL

- Copy the RSS feed URL.

![Step 7](/static/blog/94/007.png)

### Step 8: Adding Upwork RSS Feed to RSS Feed Reader

- Click on the RSS feed reader extension icon in the Edge toolbar.
- Click on the "+" button or "Add feed" option.
- Paste the copied Upwork RSS feed URL into the designated field.
- Optionally, organize the feed into a folder for better management.

![Step 8](/static/blog/94/008.png)
